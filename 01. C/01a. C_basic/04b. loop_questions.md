```C

int main() {

```

### Question 4:
- Explain output of the following program
```C
#include <stdio.h>

int main() {
    for (int i = 0; ; i++) {
        if (i >= 5)
            break;
        printf("%d\t", i);
    }
    return 0;
}
```
- Output
```
0       1       2       3       4
```

### Question 5:

### Question 8:
- Look at the following program. Why are all numbers printed in 1 line?

```C
#include <stdio.h>

int main() {
    for (int i = 2; i > 0; i--) 
        for (int j = 2; j > 0; j--)
            printf("%d %d\t", i, j);
        printf("\n");
    return 0;
}
```

- Output:
```
2 2     2 1     1 2     1 1
```
